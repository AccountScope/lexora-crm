🔧 SETUP INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════

1. RUN DATABASE MIGRATION
   $ cd /data/.openclaw/workspace/lexora
   $ psql $DATABASE_URL -f database/migrations/021_ai_features.sql

2. ENABLE PGVECTOR EXTENSION
   For Supabase: Enable via dashboard (Database > Extensions)
   For self-hosted: CREATE EXTENSION IF NOT EXISTS vector;

3. SET ENVIRONMENT VARIABLES (optional - can use UI)
   OPENAI_API_KEY=sk-...
   ANTHROPIC_API_KEY=sk-ant-...
   OLLAMA_ENDPOINT=http://localhost:11434

4. BUILD UI PAGES
   Use stubs provided in:
   • PHASE_3B_IMPLEMENTATION.md (full example for settings page)
   • components/ai/*.tsx (component stubs)

5. TEST WITH REAL DATA
   $ npm run dev
   Navigate to /settings/ai (once built)

6. (OPTIONAL) INSTALL OLLAMA FOR LOCAL AI
   $ curl -fsSL https://ollama.ai/install.sh | sh
   $ ollama pull llama3.1:8b
   $ ollama serve
